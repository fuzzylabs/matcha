matcha-ml |version| documentation

matcha-ml is your one-stop-shop for efficiently provisioning MLOps tooling and for well-structured ML workflows.

What does matcha-ml enable?
===========================

* To easily get up and running with MLOps environment where you can train and deploy models in the cloud (Azure) using only open source tooling.
* To build your own workflow to run in this environment using templates that guide you towards good MLOps by default.
